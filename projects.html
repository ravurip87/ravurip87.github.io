Projects

My work in AI, Cloud Computing, and Data Science

Cloud Computing Optimization

As part of a research mentorship program, I collaborated with a team on a cloud computing optimization project aimed at improving security and resource management using machine learning. We developed an Intrusion Detection System (IDS) to strengthen cloud security, applying machine learning algorithms to monitor network traffic and detect anomalies. Additionally, we implemented predictive models for dynamic resource allocation, which helped optimize cloud infrastructure utilization. Our efforts included designing intelligent auto-scaling strategies and using real-time demand forecasting to reduce costs while maintaining high performance and security standards.

Business Analytics Platform

As part of the OSU Business Analytics Club project, I helped develop a quantitative analysis tool for market data using Python and C++. This platform integrated various financial datasets to perform statistical modeling and trend analysis, providing insights on stock performance, market volatility, and pricing patterns. The tool featured automated data ingestion and preprocessing, advanced statistical techniques for trend forecasting, and visualization capabilities for easy interpretation of results. My contributions included designing efficient data structures in C++ to handle large datasets and implementing machine learning models in Python for predictive analysis, allowing users to make informed decisions based on data-driven insights.

Explainable Natural Language Processing for Legal Document Analysis

This project is also part of my research mentorship program, where I was a mentee. The project is designed to create an NLP system that can analyze, classify, and interpret legal documents using deep learning models. The project involves preprocessing legal text data using NLTK and SpaCy for tokenization and named entity recognition, followed by fine-tuning a BERT model to classify clauses and predict case outcomes. Healthcare Decision Support System Using Federated Learning

This project, part of my research mentorship program, aims to develop a privacy-preserving AI framework that enables hospitals to collaboratively build a diagnostic model without sharing sensitive patient data. Using TensorFlow Federated and PySyft, the system implements a federated learning setup to train models like LSTMs or CNNs on distributed electronic health records (EHR) and medical image data. Differential privacy techniques are used to protect individual patient data, ensuring that only encrypted model parameters are shared and aggregated across hospital servers. Secure aggregation methods are employed to combine updates from multiple institutions, resulting in a robust global model that performs well across diverse datasets. The final system is tested on real-world healthcare data to provide accurate disease predictions, offering a decentralized solution for diagnostic support while maintaining patient privacy.
